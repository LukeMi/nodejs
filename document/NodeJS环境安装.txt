
1.NodeJS 下载
	https://nodejs.org/en/
	（可能需要配置环境变量）
	
2.npm 安装：
	参考：http://www.runoob.com/nodejs/nodejs-npm.html

3.cnpm 安装：
	安装npm后执行：npm install -g cnpm --registry=https://registry.npm.taobao.org	
	
4.anywhere Js 安装：
	npm install anywhere -g	
	
5.vue.js 安装：
	npm install vue

6.express 安装：
	cnpm install express --save
